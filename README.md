# 🚀 Trackerly

A simple and lightweight internship application tracker built with Python and Flask.

Trackerly helps students and job seekers organize their internship and job applications, track progress, manage deadlines, and keep everything in one place.

---

## ✨ Features

- 📌 Add internship and job applications
- 🏢 Track company details
- 📅 Store application dates
- 📊 Monitor application status
- 🔍 Search and manage opportunities easily
- 💾 Local data storage
- 🌐 Browser-based desktop interface
- ⚡ Lightweight and easy to use

---

## ⚡ Quick Start

### 1. Clone the Repository

```bash
git clone https://github.com/arup-matabber/Trackerly.git
cd Trackerly
```

### 2. Install Dependencies

```bash
pip install flask
```

### 3. Run the Application

```bash
python app.py
```

### 4. Open in Browser

Visit:

```text
http://127.0.0.1:5765
```

---

## ▶️ Running Trackerly

### Windows

Run:

```bash
run.bat
```

or

```bash
python app.py
```

### macOS / Linux

Run:

```bash
chmod +x run.sh
./run.sh
```

or

```bash
python3 app.py
```

### Stop the Application

Press:

```text
Ctrl + C
```

in the terminal window.
